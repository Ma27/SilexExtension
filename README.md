SilexExtension
==============

[![Build Status](https://travis-ci.org/Ma27/SilexExtension.svg?branch=master)](https://travis-ci.org/Ma27/SilexExtension)

A simple extension for silex

Installation
------------

    {
        "require": {
            "php": ">=5.3",
            "ma27/silex-extension": "dev-master"
        }
    }

Run:

    php composer.phar install
